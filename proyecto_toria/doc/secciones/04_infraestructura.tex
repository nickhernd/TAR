\section{Infraestructura y Dependencias}
Un robot autónomo no opera de forma aislada; requiere una infraestructura de soporte integrada en el hospital.

\subsection{Estaciones de Recarga y Mantenimiento}
Se deben disponer salas específicas para la recarga automática de baterías mediante estaciones de acoplamiento (\textit{docking stations}). Estas áreas deben estar ventiladas y contar con sistemas de protección contra incendios.

\subsection{Protocolos de Desinfección}
El robot debe pasar por ciclos de desinfección programados. Se plantean dos enfoques:
\begin{enumerate}
    \item \textbf{Salas de Desinfección UV-C:} El robot entra en una sala estanca donde se somete a radiación ultravioleta para eliminar carga viral.
    \item \textbf{Sistemas Integrados:} El propio robot puede portar lámparas UV-C para desinfectar el entorno mientras se desplaza en horas de baja actividad.
\end{enumerate}
