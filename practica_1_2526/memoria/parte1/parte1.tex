\section{Servicios}
%--------------------------------------------------------------

\subsection*{Pregunta 6}
\textit{¿Qué puedes comentar acerca de este servicio?}

El servicio \texttt{add\_two\_ints} (de tipo \texttt{example\_interfaces/srv/AddTwoInts}) implementa una arquitectura cliente-servidor síncrona. El servidor queda a la espera de peticiones y el cliente envía dos enteros, recibiendo su suma como respuesta.

\begin{figure}[H]
  \centering
  \includegraphics[width=0.85\textwidth]{parte1/imagenes/servicio_suma.png}
  \caption{Ejecución del servicio suma original.}
  \label{fig:p1_servicio}
\end{figure}

\subsection*{Pregunta 7}
\textit{¿Qué servicios aparecen? ¿Cómo se ha creado el servicio \texttt{minimal\_service}?}

Al ejecutar \texttt{ros2 service list} aparece \texttt{/add\_two\_ints} junto con los servicios de parámetros del nodo (\texttt{/minimal\_service/get\_parameters}, etc.). El \texttt{minimal\_service} se crea con:
\begin{lstlisting}[style=bash]
self.srv = self.create_service(AddTwoInts, 'add_two_ints', self.callback)
\end{lstlisting}
Su cometido es sumar (o en la versión modificada, calcular la operación aritmética sobre) los valores recibidos y devolver el resultado.

\subsection*{Pregunta 8}
\textit{¿Cómo puedo saber los parámetros de la petición y la respuesta? ¿De qué tipo son?}

Con el comando:
\begin{lstlisting}[style=bash]
ros2 interface show example_interfaces/srv/AddTwoInts
\end{lstlisting}
Se obtiene: request con \texttt{int64 a} e \texttt{int64 b}; response con \texttt{int64 sum}. Son valores enteros de 64 bits.

\subsection{Ejercicio 2: Modificar el servicio suma}

Se modifica el servicio para calcular $a + b - c \cdot d \div e$, con $a, b, c, d, e \in \texttt{int64}$ y resultado en \texttt{float64}.

\textbf{Interfaz modificada} (\texttt{AddTwoInts.srv}):
\begin{lstlisting}[caption={Nueva interfaz del servicio.}]
int64 a
int64 b
int64 c
int64 d
int64 e
---
float64 sum
\end{lstlisting}

\textbf{Callback del servidor}:
\begin{lstlisting}[style=python, caption={Operación modificada en el servidor.}]
def add_two_ints_callback(self, request, response):
    response.sum = float(request.a + request.b
                         - request.c * request.d / request.e)
    return response
\end{lstlisting}
