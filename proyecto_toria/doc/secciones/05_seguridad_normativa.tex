\section{Seguridad y Normativa}
La operación de robots móviles en presencia de humanos vulnerables (pacientes) exige el cumplimiento de normativas internacionales estrictas.

\subsection{Normativa Vigente}
La norma \textbf{ISO 13482:2014} es el estándar principal para robots de cuidado personal, que incluye a los robots móviles de servicio. Esta norma define:
\begin{itemize}
    \item Límites de velocidad y fuerza en caso de colisión.
    \item Requerimientos para la detección de personas.
    \item Protocolos de parada de emergencia.
\end{itemize}

\subsection{Consideraciones Éticas y de Privacidad}
El uso de cámaras para la navegación plantea retos de privacidad. Es imperativo implementar sistemas de anonimización de rostros en tiempo real si los datos se procesan fuera del robot.
